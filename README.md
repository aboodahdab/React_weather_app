# React_weather_app
### A weather app made with React and Typescript
## How to run:
### First clone this repo using:```git clone git@github.com:aboodahdab/React_weather_app.git```
### Then go to the main folder using:```cd React_weather_app1```
### Next run ```npm install``` to install needed packages
### After that run tailwind CLI using: ``` npx @tailwindcss/cli -i ./src/index.css -o ./src/output.css --watch```
### Next run ```npm run dev``` to run the project
### Finally open your browser at port ```localhost:5173```
